Поля запроса

Section titled “Поля запроса”
ПараметрТипОбязательныйОписание
fromstring (YYYY-MM-DD)✅Начало периода выгрузки
tostring (YYYY-MM-DD)✅Конец периода выгрузки
statusstring❌Статус ордера
takeinteger❌Количество записей на страницу (по умолчанию: 100, максимум: 1000)
pageinteger❌Номер страницы (по умолчанию: 1)

Особенности:

  • Сортировка по умолчанию — по дате создания
  • Рекомендованная частота вызовов — не более 1 запроса в секунду на один API-ключ
  • Разрешённый период выгрузки — не более одного дня за один запрос
